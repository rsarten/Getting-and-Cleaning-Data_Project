## File structure
This script requires that the necessary files sit in the working directory in the following structure:

 ```
    ├── run_analysis.R
    ├── activity_labels.txt
    ├── features.txt
    ├── test
    │   └── subject_test.txt
    │   └── X_test.txt
    │   └── y_test.txt 
    ├── train
    │   └── subject_train.txt
    │   └── X_train.txt
    │   └── y_train.txt    
```
